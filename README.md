# fsh-local-api

An API solution which students can work against locally to learn automated API tests.

## Purpose

Allow students at Frans Schartau Business Institute (FSH) to create testautomation against a local running API with some endpoints.

## Theme of API

Theme of API is up to you to create, it can be anything from cars to Pokemon.

## Programming language

TypeScript/JavaScript or Python.

## Testing framework

If TypeScript/JavaScript is used:

Vitest should be prepared but not include much tests.
Supertest can also be considered as an extension.

If Python is used:

Pytest with requests

It's for the students to create the tests.
